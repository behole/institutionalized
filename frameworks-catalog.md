# Institutional Reasoning - Frameworks Catalog

Complete catalog of human decision-making systems to implement as LLM tools.

## Priority Tier (MVP - First 5)

### 1. Courtroom ⭐
**Source:** Legal system
**Use case:** Binary decisions under uncertainty
**Roles:** Prosecutor, Defense, Jury (5), Judge
**Output:** Guilty/Not Guilty + rationale + actions
**Status:** Proof of concept exists (Falconer)

**Best for:**
- Should I merge this PR?
- Publish this essay now?
- Deploy this feature?
- Accept this proposal?

### 2. Peer Review with Rebuttal ⭐
**Source:** Academic publishing
**Use case:** Validation with author response
**Roles:** Reviewers (2-4), Author (rebuttal), Editor
**Output:** Accept / Revise / Reject + specific changes

**Best for:**
- Technical documentation review
- Research proposal validation
- Spec review with feedback loop

### 3. Red Team / Blue Team ⭐
**Source:** Military security testing
**Use case:** Adversarial stress-testing
**Roles:** Blue (defender), Red (attacker), Observer
**Output:** Vulnerabilities found + severity + mitigations

**Best for:**
- Security review
- Architecture stress-testing
- Finding edge cases
- Breaking assumptions

### 4. Pre-mortem ⭐
**Source:** Business risk management
**Use case:** Identify failure modes before commitment
**Roles:** Multiple pessimists, Facilitator
**Output:** Ranked failure scenarios + mitigation strategies

**Best for:**
- Launch decisions
- Strategic planning
- Project kickoffs
- High-stakes commitments

### 5. Studio Critique ⭐
**Source:** Art/design education
**Use case:** Creative work evaluation
**Roles:** Peers, Instructor, (Creator silent initially)
**Output:** What works / What doesn't / Questions

**Best for:**
- Essay/article review
- Design review
- Creative work feedback
- UX evaluation

## Academic Frameworks

### 6. PhD Defense
**Source:** Doctoral examination
**Use case:** Deep proposal validation
**Roles:** Candidate, Committee (4-6 specialists), Chair
**Output:** Pass / Revisions / Fail + specific improvements

**Best for:**
- Complex technical proposals
- System design validation
- Research direction approval

### 7. Dissertation Committee
**Source:** Graduate education
**Use case:** Multi-stage work validation
**Roles:** Advisor, Committee members (different specialties)
**Output:** Iterative approval at milestones

**Best for:**
- Long-term project validation
- Multi-phase work review
- Thesis-level technical writing

### 8. Grant Review Panel
**Source:** Research funding
**Use case:** Comparative prioritization under resource constraints
**Roles:** Reviewers (independent scoring), Panel (calibration)
**Output:** Ranked proposals + funding line

**Best for:**
- Feature prioritization
- Resource allocation
- Competing proposals
- Portfolio optimization

## Military & Intelligence Frameworks

### 9. After-Action Review (AAR)
**Source:** Military training
**Use case:** Learning from execution
**Roles:** Participants, Facilitator
**Output:** What happened vs. expected + learnings + next time

**Best for:**
- Post-mortems
- Incident analysis
- Process improvement
- Blameless retrospectives

### 10. Intelligence Analysis (Competing Hypotheses)
**Source:** CIA analytical methods
**Use case:** Diagnostic reasoning under uncertainty
**Roles:** Analysts (generate hypotheses), Evaluator
**Output:** Ranked hypotheses + discriminating evidence

**Best for:**
- Root cause analysis
- Debugging complex systems
- Diagnostic troubleshooting
- Mystery solving

### 11. War Gaming
**Source:** Military planning
**Use case:** Scenario testing
**Roles:** Forces (multiple sides), Control, Observers
**Output:** Outcomes under different strategies

**Best for:**
- Strategic planning
- Competitive analysis
- Scenario modeling

## Business & Management Frameworks

### 12. Six Thinking Hats
**Source:** Edward de Bono
**Use case:** Multi-perspective analysis
**Roles:** 6 hats (Facts, Emotions, Critical, Optimistic, Creative, Process)
**Output:** Comprehensive analysis from all angles

**Best for:**
- Decision analysis
- Breaking out of single perspective
- Team alignment on complex issues

### 13. Devil's Advocate (Formal)
**Source:** Catholic canonization / business adapted
**Use case:** Challenge best-case assumptions
**Roles:** Proposer, Devil's Advocate (required opposition), Arbiter
**Output:** Tested proposal + addressed objections

**Best for:**
- Challenging optimistic plans
- Testing conviction
- Avoiding groupthink

### 14. Delphi Method
**Source:** RAND Corporation forecasting
**Use case:** Expert consensus building
**Roles:** Experts (anonymous), Facilitator
**Output:** Converged estimates + reasoning + outliers

**Best for:**
- Forecasting
- Estimation tasks
- Expert consensus
- Uncertainty quantification

### 15. SWOT Analysis (Structured)
**Source:** Strategic planning
**Use case:** Situational assessment
**Roles:** Internal analyst, External analyst, Synthesizer
**Output:** Strengths, Weaknesses, Opportunities, Threats + strategy

**Best for:**
- Market analysis
- Competitive positioning
- Strategic planning

**Roles:** Economic, Social, Environmental analysts, Stakeholders

- Policy analysis

## Common Patterns Across Frameworks

### Structural Elements
- **Roles** with distinct perspectives/responsibilities
- **Evidence requirements** (explicit, verifiable)
- **Synthesis mechanism** (how perspectives combine)
- **Decision format** (binary, ranked, categorical, narrative)
- **Iteration support** (single-shot vs. multi-round)

### Configuration Parameters
- **Number of agents** per role
- **Model selection** per role (temperature, reasoning depth)
- **Thresholds** for decisions
- **Rounds** of iteration
- **Evidence strictness** (loose vs. rigorous validation)

### Observability
- **Audit trail** of all agent outputs
- **Cost tracking** per framework run
- **Confidence scores** where applicable
- **Replay capability** for debugging
- **Export formats** (JSON, markdown, PDF)

## Framework Selection Guide

**For binary decisions:** Courtroom, Devil's Advocate
**For validation:** Peer Review, PhD Defense, Studio Critique
**For risk:** Pre-mortem, Red/Blue Team, War Gaming
**For diagnosis:** Differential Diagnosis, Intelligence Analysis
**For consensus:** Delphi, Consensus Circle, Tumor Board
**For creativity:** Studio Critique, Writers' Workshop, Design Critique
**For strategy:** Parliamentary, Six Hats, SWOT
**For learning:** AAR, Socratic Method
**For synthesis:** Hegelian, Talmudic
