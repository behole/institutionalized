Get Started in Seconds

Requires Bun and an API key (Anthropic, OpenAI, or OpenRouter)

Clone the repository

git clone https://github.com/yourusername/institutional-reasoning.git && cd institutional-reasoning

Install dependencies

bun install

Set your API key

export ANTHROPIC_API_KEY=your_key_here

Run a framework

bun cli.ts courtroom case.json
